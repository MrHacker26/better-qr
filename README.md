# Better-QR

Better-QR is a modern, developer-first QR code platform focused on **permanence, ownership, and flexibility**.

It allows users to generate QR codes instantly or create an account to manage a collection of **permanent, controllable QR codes** backed by a modern TypeScript stack.

This project is built as a hobby project with production-grade architecture and tooling.
